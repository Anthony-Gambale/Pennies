# Pennies
Programs for verifying conjectures about piles of pennies.

```
0 [ 1 2 3 ]
1 |  [ 2 4 ]
2 |  |  [ 1 5 ]
3 |  |  |  [ 1 1 1 1 2 ]
3 |  |  |  [ 6 ]
4 |  |  |  |  [ 1 1 1 1 1 1 ]
2 |  |  [ 1 1 1 3 ]
3 |  |  |  [ 2 2 2 ]
4 |  |  |  |  [ 3 3 ]
5 |  |  |  |  |  [ 1 1 4 ]
6 |  |  |  |  |  |  [ 1 1 2 2 ]
```
